# DTOs & Endpoints

[GET /api/recipes/{id}]
RecipeResponse
    - (int) RecipeID
    - (string) RecipeName
    - (List<IngredientResponse>) Ingredient
    - (List<InstructionResponse>) Instructions
    - (List<string>) Categories
    - (List<string>) Tags

[POST /api/recipes]
CreateRecipeRequest
    - (string) RecipeName
    - (List<IngredientResponse>) Ingredients
    - (List<InstructionResponse>) Instructions
    - (List<string>) Categories
    - (List<string>) Tags 

[PUT /api/recipes/{id}]
UpdateRecipeRequest
    - (string) RecipeName
    - (List<IngredientResponse>) Ingredients
    - (List<InstructionResponse>) Instructions
    - (List<string>) Categories
    - (List<string>) Tags

[DELETE /api/recipes/{id}]
NO NEED DTO

[not directly part of an endpoint]
IngredientResponse
    - IngredientName
    - Quantity
    - Unit

[not direclty part of an endpoint]
InstructionResponse
    - StepNumber
    - Description

# C# Models

Recipe
    - (int) RecipeID, PK
    - (string) RecipeName
    - (ICollection<RecipeIngredient>) Ingredients
    - (ICollection<RecipeInstructionStep>) Instructions
    - (ICollection<Category>) Categories
    - (ICollection<Tag>) Tags

RecipeIngredient
    - (int) RecipeID, FK
    - (int) RecipeIngredientID, PK
    - (Recipe) Recipe
    - (string) Name 
    - (decimal) Quantity
    - (string) Unit

RecipeInstructionStep
    - (int) RecipeID, FK
    - (int) RecipeInstructionID, PK
    - (Recipe) Recipe
    - (int) StepNumber
    - (string) Description

Category
    - (int) CategoryID 
    - (string) CategoryName
    - (ICollection<Recipe>) Recipes

Tag
    - (int) TagID 
    - (string) TagName 
    - (ICollection<Recipe>) Recipes
